# 小猫图片

猫按**体型**分组命名(前缀即体型):

| 前缀 | 体型 | 现有文件 | 说明 |
|---|---|---|---|
| `l`  | 大 (large)  | `l1 l2 l3`            | 画得更大;每题固定 1 只 |
| `m`  | 中 (medium) | `m1 m2 m3 m4 m5`     | 常规大小 |
| `sp` | 中·特殊     | `sp1 sp2 sp3 sp4`    | 也按中等处理 |
| `s`  | 小 (small)  | `s2 s3`              | 画得更小;每题固定 1 只 |

- 每题随机抽取:**1 只大猫 + 1 只小猫 + 其余中猫**,凑够网格边长(6×6 用 6 只 … 9×9 用 9 只)。
- 大猫 / 小猫各只有 1 只,所以线索能说 **"次卧的书桌上蹲着一只大猫"** 直接把它钉死。
- **猫的名字 = 文件名**(去掉 `.png`)。线索里就叫 `l2`、`m3`…… 想要好看的名字,直接把文件重命名(如 `橘子.png`),游戏里自动显示 `橘子`,不用改代码。
- 缺图会回退到 emoji 猫脸,不报错。

## 增删猫或改体型

编辑 [`web/src/theme/catdoku.js`](../../src/theme/catdoku.js) 里的 `CAT_POOL`
(按 large/medium/small 三组列出文件名),以及 `CAT_ORDER`/`CAT_COLORS`
(给每只猫一个格子底色)。生成器只需保证每题 1 大 1 小,其余中等。
