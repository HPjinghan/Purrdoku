# 小猫图片放这里

把猫的图片放进这个目录,按**编号**命名:

```
cat1.png  cat2.png  cat3.png  cat4.png  cat5.png
cat6.png  cat7.png  cat8.png  cat9.png
```

- **共 9 张**(编号 1–9)。6×6 的题用到 1–9 中随机 6 只,9×9 用到全部 9 只。
- **格式**:建议 **PNG、透明背景、正方形**(如 256×256 或 512×512);会被裁成圆形显示。
- 每张图对应一个固定编号:游戏里这只猫就一直用这张图 + 这个编号角标。
- **没放图也能玩**:缺哪张就自动回退到对应的 emoji 猫脸,不会报错。

> 想换扩展名(如 `.jpg`/`.webp`)或换目录,改 `web/src/theme/catdoku.js` 里的 `catImage()` 一行即可。
